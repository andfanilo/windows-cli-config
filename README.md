# Command Line Config

Modern CLI setup protocols for my development machines. Both use the same core stack: **Starship + Neovim + fzf + ripgrep + fd + zoxide + eza + bat + delta**.

## Protocols

- **[WINDOWS.md](WINDOWS.md)** — Windows 11 + Windows Terminal Preview + PowerShell 7 + Scoop
- **[UBUNTU.md](UBUNTU.md)** — Ubuntu 24.04 LTS + Zsh + Homebrew + tmux

## Shared Config

The following configs are identical across both machines:

- **Starship prompt** (`~/.config/starship.toml`)
- **Neovim** (kickstart.nvim)
- **Git delta** settings
- **FZF** defaults and keybindings
